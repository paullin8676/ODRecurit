# 系统使用文档

## 1. 系统概述

本系统是一个招聘流程管理系统，用于管理候选人的整个招聘流程，包括考试、面试等环节。系统分为前端和后端两部分，前端使用Vue 3框架，后端使用Express.js和SQLite数据库。

### 1.1 系统功能

- 用户认证与授权
- 候选人管理
- 产品线管理
- 考试试卷管理
- 考试类型管理
- 考试及格线管理
- 面试阶段管理
- 数据统计分析

## 2. 系统登录

1. 打开浏览器，访问系统地址
2. 进入登录页面，输入用户名和密码
3. 点击登录按钮，进入系统主界面

## 3. 系统界面

### 3.1 仪表板

仪表板是系统的主界面，显示系统的概览信息，包括：
- 候选人总数
- 各阶段候选人数量
- 招聘流程统计图表

### 3.2 候选人管理

候选人管理页面用于查看和管理所有候选人信息：
- 查看候选人列表
- 查看候选人详细信息
- 编辑候选人信息
- 查看候选人的考试和面试记录

### 3.3 用户管理

用户管理页面用于管理系统用户：
- 查看用户列表
- 添加新用户
- 编辑用户信息
- 删除用户

### 3.4 产品线管理

产品线管理页面用于管理产品线信息：
- 查看产品线列表
- 添加新产品线
- 编辑产品线信息
- 删除产品线

### 3.5 考试试卷管理

考试试卷管理页面用于管理考试试卷：
- 查看试卷列表
- 添加新试卷
- 编辑试卷信息
- 删除试卷

### 3.6 考试类型管理

考试类型管理页面用于管理考试类型：
- 查看考试类型列表
- 添加新考试类型
- 编辑考试类型信息
- 删除考试类型

### 3.7 考试及格线管理

考试及格线管理页面用于管理考试及格线：
- 查看及格线列表
- 添加新及格线
- 编辑及格线信息
- 删除及格线

### 3.8 数据统计

数据统计页面用于查看系统的统计数据：
- 招聘漏斗分析
- 各阶段转化率
- 招聘周期分析
- 考试成绩分布

## 4. 系统操作流程

### 4.1 添加候选人

1. 进入候选人管理页面
2. 点击添加按钮
3. 填写候选人信息
4. 选择产品线
5. 点击保存按钮

### 4.2 安排考试

1. 进入候选人详细信息页面
2. 点击安排考试按钮
3. 选择考试类型和试卷
4. 点击保存按钮

### 4.3 记录面试

1. 进入候选人详细信息页面
2. 点击记录面试按钮
3. 填写面试信息和评价
4. 点击保存按钮

### 4.4 审批流程

1. 进入候选人详细信息页面
2. 查看候选人的考试和面试记录
3. 根据记录做出审批决定
4. 更新候选人状态

## 5. 系统配置

### 5.1 用户权限

系统用户分为不同角色，具有不同的权限：
- 管理员：具有所有操作权限
- 招聘专员：可以管理候选人信息和面试流程
- 面试官：可以查看候选人信息和记录面试评价

### 5.2 系统设置

- 考试及格线设置
- 面试流程设置
- 通知设置

## 6. 常见问题

### 6.1 登录问题

- 忘记密码：点击忘记密码链接，按照提示重置密码
- 账号锁定：联系管理员解锁账号

### 6.2 操作问题

- 无法保存数据：检查网络连接，确保输入信息完整
- 页面加载缓慢：检查网络连接，清除浏览器缓存

### 6.3 数据问题

- 数据丢失：联系管理员恢复数据
- 数据错误：联系管理员修正数据
