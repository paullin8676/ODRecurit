# 招聘管理导航菜单修改规格说明

## Why
需要将一级菜单"员工管理"改名为"招聘管理"，更好地反映该菜单的功能：按阶段来管理候选人在各阶段的记录。

## What Changes
- 将一级菜单"员工管理"改名为"招聘管理"
- 二级菜单保持不变：候选录入、机考管理、韧测管理、面试管理、员工管理

## 功能说明

### 招聘管理菜单结构
- **招聘管理** (一级菜单)
  - 候选录入 (/employees) → 候选录入模块配置
  - 机考管理 (/exam-stage) → 机考管理模块配置
  - 韧测管理 (/test-stage) → 韧测管理模块配置
  - 面试管理 (/interview-stage) → 面试管理模块配置
  - 员工管理 (/employee-management) → 员工管理模块配置

### 核心功能
每个二级菜单页面只显示该模块配置中勾选的阶段的候选人记录：
- 候选录入页面 → 只显示阶段配置中"候选录入模块"勾选的阶段
- 机考管理页面 → 只显示阶段配置中"机考管理模块"勾选的阶段
- 韧测管理页面 → 只显示阶段配置中"韧测管理模块"勾选的阶段
- 面试管理页面 → 只显示阶段配置中"面试管理模块"勾选的阶段
- 员工管理页面 → 只显示阶段配置中"员工管理模块"勾选的阶段
